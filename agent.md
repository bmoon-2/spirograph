# agent.md

## 目的
このリポジトリ（Shape Spirograph）で作業するエージェント向けの運用ガイド。

## プロジェクト概要
- `index.html`: UI とスクリプト読込順の定義。
- `style.css`: レイアウトと見た目。
- `js/state.js`: 共有状態の保持。
- `js/ui.js`: UI 入出力とイベント処理。
- `js/contour.js`: 文字/図形輪郭の生成。
- `js/math.js`: スピログラフ計算ロジック。
- `js/draw.js`: Canvas 描画処理。
- `js/debug.js`: デバッグ表示の制御。
- `js/export.js`: SVG/PNG 書き出し。
- `js/main.js`: 初期化と全体フロー。

## エージェントの役割
- 既存機能を壊さずに、最小差分で改善する。
- UI 文言は既存の日本語トーンに合わせる。
- バグ修正時は再現手順と確認手順を明確にする。

## 作業ルール
- 変更前に対象ファイルを読み、依存関係（特に `index.html` の読込順）を確認する。
- 影響範囲が狭い場合は局所修正を優先する。
- 無関係な整形・リネームはしない。
- 文字コードは UTF-8 を維持する。
- 外部ライブラリは、明示的な依頼がない限り追加しない。

## 実装ガイド
- 新しい状態値は `js/state.js` に追加し、参照箇所を更新する。
- UI コントロール追加時は `index.html` と `js/ui.js` をセットで更新する。
- 描画仕様の変更は `js/math.js` と `js/draw.js` の責務を分離したまま実装する。
- Export 形式の変更は `js/export.js` に閉じる。

## 確認チェックリスト
- 描画開始/停止が機能する。
- 各スライダー（ギア半径・ペン距離・周回数・スピード）が反映される。
- 色モード切替が反映される。
- デバッグ表示の ON/OFF が機能する。
- `clear` 後に再描画できる。
- SVG/PNG の書き出しが動作する。
- コンソールエラーが発生しない。

## 完了条件
- 依頼内容が実装され、上記チェックで影響範囲を確認済み。
- 変更ファイルと意図を簡潔に説明できる状態。
